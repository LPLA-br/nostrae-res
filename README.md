## DESCRIÇÃO

<p>
    Servidor RESTful para o front-end React da aplicação nostrae-res
</p>


## Instalação

```bash
$ npm install
```

## Execução da aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes do software criado

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
